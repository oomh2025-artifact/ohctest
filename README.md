# 労働衛生コンサルタント 口述試験対策アプリ

労働衛生コンサルタント（保健衛生区分）の口述試験対策用チャットボットアプリです。

## 🎯 機能

### 本編（詳細解説付き）
- 基本→詳細→応用の3段階構成
- 各問題に詳細な追加質問と回答付き
- 5点以下の場合、補足質問でリカバリーのチャンス

### 追加問題編
- 補足的な問題で知識を補強
- 本編で学んだ内容を別角度から確認

### カスタム問題
- アプリ内で問題を追加可能
- Markdown形式で入力
- JSON形式でのインポート/エクスポート対応

### 共通機能
- 🎯 ランダム出題（5テーマ）
- 🤖 AI採点（10点満点、講評付き）
- 📦 持ち帰り機能（2回まで）
- 📚 復習フォルダ（5点以下を自動保存）
- ✅ 合格判定（60%以上で合格）

## 📁 ファイル構成

```
├── index.html              # アプリ本体
├── data/
│   ├── questions_main.md   # 本編（詳細解説付き）
│   ├── questions_extra.md  # 追加問題編
│   └── followup.md         # 追加質問回答集
└── README.md
```

## 🔧 セットアップ

### APIキーの設定
AI採点機能を使用するには、Anthropic APIキーが必要です。

1. [Anthropic Console](https://console.anthropic.com/settings/keys) でAPIキーを取得
2. アプリのメニュー画面で「APIキー設定」から入力
3. キーはブラウザに保存され、次回以降は自動読み込み

## 📝 問題の追加・編集

### アプリ内での問題追加（カスタム問題）

1. メニュー画面で「➕ 問題追加」をタップ
2. Markdown形式で問題を入力（テンプレートボタンあり）
3. 「🔍 プレビュー」でパース結果を確認
4. 「💾 カスタム問題として保存」で保存

追加した問題はブラウザに保存され、次回以降も使用可能です。

### JSON形式でのインポート/エクスポート

- **エクスポート**: プレビュー画面で「📋 JSONコピー」をクリック
- **インポート**: 「JSONインポート」タブでJSON形式のデータを貼り付け

## 📊 採点基準

試験官の4つの評価軸に基づいて採点：

1. **知識の深さ** - 暗記ではなく理解しているか
2. **実務での対応力** - 現場でコンサルティングできるか
3. **判断の誠実さ** - わからないことを正直に言えるか
4. **思考の柔軟性** - 想定外の質問にも対応できるか

### 点数の目安
- 10点：完璧
- 8-9点：優秀
- 6-7点：合格
- 4-5点：一部正解
- 2-3点：方向性のみ
- 0-1点：的外れ
